# App Blueprint Prompt for ai-todo-list

Use this prompt to regenerate or rewrite the ai-todo-list app from scratch, preserving its core features and structure. This blueprint ensures you can recreate the app or adapt it for similar projects.

---

**Prompt:**

> Rewrite the ai-todo-list app as a single-page HTML application. The app should:
> 
> - Allow users to add, edit, delete, and reorder todo items (with both drag-and-drop and arrow buttons).
> - Support inline editing of todos.
> - Provide a Save button to store the todo list in local storage, with selectable expiry options (1, 7, 30, or 90 days).
> - Provide a Remove button to clear the todo list from local storage.
> - On page load, automatically load the todo list from local storage if it exists and is not expired.
> - Feature a clean, modern, responsive UI with card-style layout and accessible controls.
> - All code (HTML, CSS, JavaScript) should be in a single file (index.htm).
> - Use semantic HTML and best practices for accessibility and maintainability.
>
